# CSS Bootstrap Use Cases

## Project Overview

This project demonstrates four professional UI use cases built with HTML5 and Bootstrap 5 CDN. Each use case is implemented in a separate HTML file using internal CSS styles, responsive layout, and modern CSS3 features suitable for a business application.

## Technologies Used

- HTML5
- Bootstrap 5 (CDN)
- CSS3 features: variables, transitions, shadows, border radius, hover effects, media queries, pseudo-classes

## Use Cases Implemented

- `navbar.html`: Responsive navigation bar with brand, menu links, and login button
- `hero.html`: Business hero section with heading, description, CTA buttons, and placeholder image
- `login.html`: Centered login form with email/password fields, remember me, and forgot password link
- `profile-card.html`: Professional profile card with circular avatar, user details, and follow button

## How to Run

1. Open the `css-bootstrap-use-cases` folder in your browser or editor.
2. Open any HTML file (`navbar.html`, `hero.html`, `login.html`, or `profile-card.html`) in a browser.
3. No server or JavaScript is required.

> All pages are fully responsive and use only HTML and CSS with Bootstrap components.
